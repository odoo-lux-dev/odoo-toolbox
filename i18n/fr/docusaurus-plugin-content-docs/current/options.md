# Options

La page Options regroupe tous les réglages de l'extension en un seul endroit. Pour l'ouvrir :

- Faites un **clic droit** sur l'icône Odoo Toolbox dans la barre d'outils de votre navigateur, puis sélectionnez **Options**.
- Ou ouvrez le [popup de l'extension](./odoo/popup) et cliquez sur le lien **Options** en bas du popup.

La plupart des fonctionnalités d'Odoo Toolbox sont activables, désactivables ou configurables depuis cette page.

:::tip
N'hésitez pas à l'explorer : c'est le meilleur moyen de vous approprier l'extension et de la personnaliser selon vos besoins.
:::

![Page Options](/img/options/overview.png)

## Export / Import de la configuration {#export-import}

La page Options permet également d'exporter et d'importer l'ensemble de votre configuration. Utile pour sauvegarder vos réglages ou les transférer vers un autre navigateur ou une autre machine.
