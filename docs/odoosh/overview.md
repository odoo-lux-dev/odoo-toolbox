# Odoo.SH

Odoo Toolbox adds a set of features directly into the Odoo.SH interface to simplify project and branch management. These features are only available on [Odoo.SH](https://www.odoo.sh).

## What's included {#whats-included}

### [Project Favorites](./favorites) {#project-favorites}

The flagship feature for Odoo.SH. Star your most-used projects, give them custom display names, and keep them sorted the way you want - so you always find what you need at a glance.

### [Additional Features](./additional-features) {#additional-features}

Utilities that improve your daily Odoo.SH workflow: branch name copy, GitHub links, task integration, colorblind-friendly build status indicators, and more.
